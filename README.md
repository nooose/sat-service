# 기술 다 때려박아 (SAT: Stuff All Tech)
커뮤니티 사이트를 만들어보자!

## 기술 스택
- Kotlin / Java 21
- Gradle
- Spring Boot 3.3.0
  - Web
  - Security
  - Data JPA
    - ~~Querydsl~~
    - Kotlin JDSL

- 테스트
  - JUnit / Kotest / Mockk
  - API 문서
    - Swagger
    - RestDocs

## Quick Start
```bash
cd ${PROJECT_DIR}
./gradlew bootJar
java -jar build/libs/application.jar
```
