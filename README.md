# 기술 다 때려박아 (SAT: Stuff All Tech)
스터디 그룹을 만들고 참여할 수 있는 공간

## 기술 스택
- Kotlin / Java 21
- Gradle
- Spring Boot 3.2.X
  - Web
  - Data JPA / Querydsl

- 테스트
  - JUnit / Kotest / Mockk
  - API 문서
    - Swagger
    - RestDocs

## Quick Start
```bash
cd ${PROJECT_DIR}
./gradlew bootJar
java -jar build/libs/application.jar
```
